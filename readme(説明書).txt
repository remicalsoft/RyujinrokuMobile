四聖龍神録2のモバイル版です。
本プロジェクトはDXライブラリAndroid版用のプロジェクトです。
https://dxlib.xsrv.jp/use/dxuse_vscom2019_android.html
こちらを確認ください。
面倒な設定は不要ですが、必要なインストールを終えている必要があります。
VisualStudioのインストーラーで
『C++ によるデスクトップ開発』『.NET によるモバイル開発』『C++ によるモバイル開発』
をインストールください。
